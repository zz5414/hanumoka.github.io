Atom 개발용 내장 웹서버 패키지(atom-live-server)

작성일 2018-11-06 Edited on 2020-04-16 In atom

들어가기

Atom개발툴로 html기반의 정적 웹을 실시간으로 그 결과를 볼수 있는 atom-live-server 라는 패키지가 있다.
단순힌 웹 개발을 할때 아주 편하게 사용 할 수 있다.

일단 atom 툴을 키고 File - Settings - Install 로 이동하여 atom-live-server 를 검색하여 설치하자.

설치를 끝낸 다음에

대충 보니 ctrl-alt-3 키드를 한꺼번에 누르면 3000번 서버로 서버를 실행하고, ctrl-alt-q를 누르면 실행중인 웹서버를 종료하는 것 같다.

ctrl-alt-3를 눌러보면

위 이미지에 동작중인 서버의 포트가 이상하게 나오는 것은 기존에 3000 포트를 사용중인 서비스가 있어서 자동으로 포트를 따준것 같다.

ctrl-alt-숫자 형식이므로 숫자를 바꿔 원하는 포트를 사용하면 될것 같다.

그리고 atom 에서 ctrl-alt-q를 눌러 서버를 종료 시키자.

위 이미지는 내가 여러번 눌러서 여러번 나온것이다.

끝!!
