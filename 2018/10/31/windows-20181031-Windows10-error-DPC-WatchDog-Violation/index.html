들어가기

Windows10 64bit pro를 데스크탑에 사용중이다.
얼마전부터 윈도우가 갑작이 프리징되고, 조금 시간이 지나면 블루스크린과 error DPC WatchDog Violation라는 에러메세지가 발생했다.

인터넷을 찾아보니 여러가지 발생이유가 있었다.
메인보드 드라이버, 바이오스
기타 드라이버 등등
