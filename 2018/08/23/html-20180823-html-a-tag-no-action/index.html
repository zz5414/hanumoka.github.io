들어가기

html a 태그(앵커)는 html 문서에서 링크를 지정하는 태그이다.
그 표시로 마우스 커서를 a 태그에 가져가면 커서가 손 모양으로 변한다.

개발하다보면, 이 a 태그의 링크 동작이 필요 없는 경우가 있다.
(javascript 라이브러리로 동적으로 a 태그를 수정하는 경우…)
